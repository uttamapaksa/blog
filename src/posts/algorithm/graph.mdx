## 1. Graph
그래프는 정점(Vertex, Node)과 정점들을 연결하는 간선(Edge)의 집합이다. 현실 세계의 관계나 구조를 추상화한 모델로서 사용된다. 종류로는 방향 그래프, 무향 그래프, 가중치 그래프, 비가중치 그래프, 부분 그래프, 순환 그래프, 비순환 그래프 등이 있다. 그래프를 통해 네트워크 관계를 표현하고, 경로를 최적화하고, 작업 스케쥴링을 할 수 있다.

## 3. DFS
깊이 우선 탐색(Depth-First Search, DFS)은 시작 정점에서 한 방향으로 끝까지 탐색하다가 더 이상 갈 곳이 없으면 가장 마지막에 만났던 갈림길 간선이 있는 정점으로 돌아가 다른 방향으로 순회하며 모든 정점을 탐색하는 방법이다. 더 이상 갈 곳이 없을 때 직전에 방문했던 곳으로 돌아가야 하기 때문에, 각 단계에서 이전 방문 정점을 후입선출 구조의 스택에 저장한다. O(V+E)의 시간 복잡도를 가진다.
```python
def dfs(u):
    stack.append(u)
    visit.append(u)
    while stack:
        u = stack.pop()
        for v in graph[u]:
            if v in visited: continue
            stack.append(v)
            visit.append(u)

graph = {1: [2, 3], 2: [1, 4], 3: [1, 5], 4: [2], 5: [3]}
stack = []
visited = []
dfs(1)

def dfs(u):
    visited.append(u)
    for v in graph[u]:
        if v not in visited:
            dfs(v)

graph = {1: [2, 3], 2: [1, 4], 3: [1, 5], 4: [2], 5: [3]}
stack = []
visited = []
dfs(1)
```

## 4. BFS
너비 우선 탐색(Breadth-First Search, BFS)은 시작 정점에서 모든 인접 정점들을 방문한 후에, 전 단계에 방문했던 모든 정점들을 출발점으로 하여 다시 모든 인접 정점들을 방문하는 방식으로 탐색하는 방법이다. 전 단계 시작점들의 모든 인접 정점을 먼저 탐색해야 하기 때문에, 각 단계에서 이전 방문 정점을 선입선출 구조의 큐에 저장한다. O(V+E)의 시간 복잡도를 가진다.
```python
from collections import deque

def bfs(u):
    queue.append(u)
    visited.append(u)
    while queue:
        u = queue.popleft()
        for v in graph[u]:
            if v not in visited:
                queue.append(v)
                visited.append(v)

graph = {1: [2, 3], 2: [1, 4], 3: [1, 5], 4: [2], 5: [3]}
queue = deque()
visited = []
bfs(1)
```

## 5. Topological Sort
위상 정렬(Topological Sort)은 방향 그래프에서 정점들이 의존하는 선후 관계를 고려하여 정점들을 나열하는 방법이다. 정점들의 부분 순서를 지키면서 전체 그래프 정점들을 정렬한다. 선후 관계는 간선을 통해 나타내며, 진입 차수가 0인 정점부터 탐색을 시작한다. O(V+E)의 시간복잡도를 가진다.
```python
from collections import deque

def topological_sort():
    indegrees = {u: 0 for u in graph}
    for u in graph:
        for v in graph[u]:
            indegrees[v] += 1
    
    queue = deque()
    for u in graph:
        if indegrees[u] == 0:
            queue.append(u)
    
    while queue:
        u = queue.popleft()
        visited.append(u)
        for v in graph[u]:
            indegrees[v] -= 1
            if indegrees[v] == 0:
                queue.append(v)
    
    if len(visited) != len(graph):
        visited.clear()

graph = {1: [2, 3], 2: [4], 3: [4], 4: [5], 5: []}
visited = []
topological_sort()
```

## 6. Dijkstra
다익스트라 알고리즘(Dijkstra Algorithm)은 간선에 양의 가중치가 있는 그래프에서, 시작 정점에서 다른 모든 정점까지의 최단 경로를 찾는 알고리즘이다. 현재까지 찾아진 최단 경로를 활용하여 더 짧은 경로를 찾아내는 간선 완화(edge relaxation)를 탐욕적(greedy)으로 수행하기 때문에, 한번 방문한 정점에 대해서는 최적해(최단 경로)가 구해져서 다시 방문할 필요가 없다. O((V+E)logV)의 시간 복잡도를 가진다.
```python
from heapq import heappush, heappop

def dijkstra(r):
    distances[r] = 0
    heappush(priority_queue, (distances[r], r))

    while priority_queue:
        d, u = heappop(priority_queue)
        
        # pruning
        if u in visited:
            continue
        visited.add(u)
        # if d > distances[u]:
        #     continue
        
        for w, v in graph[u]:
            if distances[v] > d + w:
                distances[v] = d + w
                heappush(priority_queue, (distances[v], v))

graph = {
    1: [(1, 2), (4, 3)],
    2: [(1, 1), (2, 3), (5, 4)],
    3: [(4, 1), (2, 2), (1, 4)],
    4: [(5, 2), (1, 3), (3, 5)],
    5: [(3, 4)]
}
distances = {vertex: float('inf') for vertex in graph}
priority_queue = []
visited = set()
dijkstra(1)
```

## 7. Bellman-Ford
벨만-포드 알고리즘(Bellman-Ford Algorithm)은 음의 가중치 간선이 있는 그래프에서, 시작 정점에서 다른 모든 정점까지의 최단 경로를 찾는 알고리즘이다. 정점의 개수가 V일 때 간선들을 V-1번 순회하여 최소 V-1개 이하의 간선으로 연결된 모든 최단 경로를 갱신한다. 이후 간선 완화를 한번 더 진행해서 갱신 여부를 통해 음의 사이클이 존재하는지 확인한다. O(V*E)의 시간 복잡도를 가진다.
```python
def bellman_ford(r):
    distances[r] = 0

    for _ in range(len(vertices) - 1):
        for u, v, w in graph:
            if distances[v] > distances[u] + w:
                distances[v] = distances[u] + w

    for u, v, w in graph:
        if distances[v] > distances[u] + w:
            distances.clear()
            return

graph = [
    (1, 2, -1),
    (1, 3, 4),
    (2, 3, 3),
    (2, 4, 2),
    (2, 5, 2),
    (4, 3, 5),
    (4, 2, 1),
    (5, 4, -3),
]
vertices = set()
for u, v, w in graph:
    vertices.add(u)
    vertices.add(v)
distances = {vertex: float('inf') for vertex in vertices}
bellman_ford(1)
```

## 8. Floyd-Warshall
플로이드 워셜 알고리즘(Floyd-Warshall Algorithm)은 모든 정점 간 최단 경로를 찾는 알고리즘이다. 모든 출발 정점, 도착 정점 그리고 경유 정점에 대해 간선 완화 과정을 수행한다. 특정 점정 간 최단 경로를 계산하면서 전체 정점 간 최단 경로를 구하는 동적 프로그래밍(DP) 방식을 사용한다. 음의 가중치 간선이 있는 그래프에도 적용할 수 있다. O(V^3)의 시간복잡도를 가진다.
```python
def floyd_warshall():
    for i in range(n):
        for j in range(n):
            if i == j:
                distances[i][j] = 0
            elif graph[i][j] != 0:
                distances[i][j] = graph[i][j]

    for k in range(n):
        for i in range(n):
            for j in range(n):
                if distances[i][j] > distances[i][k] + distances[k][j]:
                    distances[i][j] = distances[i][k] + distances[k][j]

graph = [
    [0, 2, 0, 4],
    [0, 0, 1, 0],
    [0, 0, 0, 3],
    [0, 0, 0, 0]
]
n = len(graph)
distances  = [[float('inf')] * n for _ in range(n)] 
floyd_warshall()
```
