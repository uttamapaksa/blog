# [My blog](https://uttamapaksa.vercel.app/)

Welcome to my personal blog. This blog contains my study notes and books I've read.

### It's built with
- **Framework :** Next.js  
- **Language :** TypeScript  
- **Styling :** Tailwind CSS, Headless UI
- **Content :** next-mdx-remote
- **Deployment :** Vercel
