# [My blog](https://uttamapaksa.github.io/)

Welcome to my personal blog. This blog contains my study notes and books I've read.

### It's built with
- **Framework :** Next.js  
- **Language :** TypeScript  
- **Styling :** Tailwind CSS, Headless UI, Heroicons  
- **Content :** gray-matter, next-mdx-remote
- **Deployment :** Vercel
